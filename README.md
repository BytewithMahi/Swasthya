## 📖 About

Swasthya is an AI-powered healthcare management platform designed for Primary Health Centres (PHCs) and Community Health Centres (CHCs).

It enables administrators to monitor healthcare resources, predict shortages using AI, and maintain trusted records using blockchain.

---

## 🚨 Problem Statement

- 💊 Medicine Stock-outs
- 🛏️ Bed Shortages
- 👥 Patient Footfall
- 👨‍⚕️ Doctor Availability
- 📊 No Real-time Monitoring
